# Projeto_FInal_de_HTML

 

Objetivo do Projeto 

Este projeto é um portfólio de página única desenvolvido como atividade do módulo de Front-End, com foco em HTML + CSS, responsividade e fidelidade ao design fornecido pelo desafio do Frontend Mentor. 

O objetivo foi construir um layout moderno que simula um site de venda de cursos de programação, com seções de apresentação, projetos, habilidades e formulário de contato. 

 Tecnologias Utilizadas 

• HTML5 

• CSS3 (Flexbox e Grid Layout) 

• Media Queries (Responsividade) 

• Reset/Normalize 

• Validação do formulário com JavaScript 

 Funcionalidades 

• Visualizar o layout ideal em diferentes dispositivos (mobile, tablet e desktop) 

• Visualizar estados de hover e foco em botões e elementos interativos 

• Interagir com o formulário de contato 

• Receber mensagens de erro se qualquer campo estiver vazio ou o e-mail estiver incorreto 

 Requisitos Atendidos 

• Layout fiel ao design proposto 

• Tipografia, espaçamento e cores seguindo o protótipo 

• Grid de projetos exibido corretamente 

• Responsividade para diversos tamanhos de tela 

• Validação de formulário no front-end 

• Estados de interação (hover e focus) 

 

 

Como Executar o Projeto 

1. Clone o repositório: 
   git clone: https://github.com/Gustavostackx/Projeto_FInal_de_HTML.git 

2. Entre na pasta: 
   cd Projeto Final de HTML 

3. Abra o arquivo no navegador: 
   index.html
